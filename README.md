## Tap on time

Игра в жанре hyper-casual, которая представляет собой кликер на реакцию. Игра имеет один бесконечный режим, который работает за счет генерации уровней разной сложности. 
За игровую валюту, которую игрок получает в конце каждого уровня, можно приобрести различные скины, которые ни как не влияют на игровой процесс.
Дополнительным источником игровой валюты являеются ежедневные задания.

## Монетизация

Монетизация в игре полностью заточена на показах рекламы. Основным источником является rewarded реклама, для продолжения уровня после проигрыша, дополнительная награда за задания и так далее.
